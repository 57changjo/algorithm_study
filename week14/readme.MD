## 📝 리뷰 요약 (최현준)
### 문제 1: (백준 27970 - OX)
- 풀이 접근: 0을 X, 1을 O로 치환하고 풀어야함
- 주요 코드/아이디어: 채워지면 올라간다는 아이디어. O를 만날 때 이전의 수를 기준으로. 
- 리뷰 포인트: Div를 어디서 나눠야 하는가? 어디서 오버플로우가 나는가?

### 문제 2: (백준 30023 - 전구 상태 바꾸기)
- 풀이 접근: 그리디
- 주요 코드/아이디어: 색깔을 숫자로 치환 -> 모두 0, 1, 2로 만드는데 드는 비용(연산 수) 계산 -> N-3d에서 모든 색깔에 대해 바꾸는 비용을 계산하고 가장 최소 비용 가져옴
- 리뷰 포인트: min함수를 사용하고 싶을 때, 임의 지정한 큰 수 보다는 INT_MAX를 사용

### 문제 3: (백준 1245 - 농장관리)
- 풀이 접근: 전형적 BFS문제, 8개 방향 봐야함
- 주요 코드/아이디어: Visited 사용, 덩어리 기준으로 봐야 함. 내가 산봉우리인지 아닌지 판단, 나보다 주변에 높은 칸 있으면 내가 산봉우리 아님 -> 내가 산봉우리인지 아닌지만 BFS에서 판단하면 됨
- 리뷰 포인트: vector로 좌표 방향 설정 가능
