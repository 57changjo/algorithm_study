## 📌 스터디 개요
- **진행 일정**: 매주 월요일 18:30 ~ 20:30
- **플랫폼**: 백준, 정올
- **사용 언어**: C++
- **진행 기간**: 2025년 9월 ~ 
- **목적**
  1. 알고리즘/자료구조 심화 학습
  2. 협업 능력 및 코드 리뷰 경험 축적

---

## 🚀 운영 방식 요약
1. 모든 인원: 문제 사전 풀이 + 코드 업로드
2. 출제자: 주차별 문제(3~4개) 선정
3. 발표자: 문제별 풀이/코드 설명
4. 리뷰: 스터디 시간 내 실시간 리뷰 + 사전 메모 기반 토론
5. 정리 담당: 스터디 후 `README.md` 업데이트

세부 규칙은 👉 [`docs/study_rules.md`](./docs/study_rules.md) 참고
